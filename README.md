## Getting Started


### Prerequisites

Python 3.8.5


### Installation

1. Create virtualenv
   ```sh
   pip install virtualenv
   ```

   ```sh
   virtualenv venv --no-site-packages
   ```
   or if virtualenv --version >= 17.0
    ```sh
   virtualenv venv
   ```

2. Start venv
   Linux/Unix:
   ```sh
   source venv/bin/activate
   ```
   Powershell:
   ```sh
   venv/Scripts/activate.ps1
   ```

2. Install PIP packages
   ```sh
   pip install -r requirements.txt
   ```
